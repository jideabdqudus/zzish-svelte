## Tech Stack

- Framework: Svelte
- Styling: Tailwind

## Installation

1. Clone the repository

2. Ensure the project runs within the desired node engine:
   `nvm use`

3. Install dependencies
   `npm install`

4. Run development server
   `npm run dev`

5. For testing
   `npm run test`
